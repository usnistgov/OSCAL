Metadata group: meta-group assembly

Custom- or application-defined annotated metadata.

The meta-group assembly has the following contents (in order):

  • term flag (optional)
  • type flag (optional)
  • meta fields (zero or more)
  • meta-group assemblies (zero or more)
  • notes assembly (zero or one)
