Control: control object

A structured information object representing a security or privacy control. Each security or privacy control within the Catalog is defined by a distinct control instance.

The control object has the following members (properties):

  • id property (required)
  • class property (optional)
  • title object (required)
  • An array labelled , containing param objects (optional)
  • An array labelled , containing prop objects (optional)
  • An array labelled , containing link objects (optional)
  • An array labelled , containing part objects (optional)
  • An array labelled , containing subcontrol objects (optional)
  • ref-list object (optional)

Controls may be grouped using group, and controls may be partitioned using part or enhanced using subcontrol.

  • {% highlight json %} { "id" : "x", "title" : "Control 1" } {% endhighlight %}

This object appears as a property on: catalog, group.
