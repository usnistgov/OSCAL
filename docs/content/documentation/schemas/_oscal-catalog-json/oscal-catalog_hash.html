Hash: hash string object

This object appears unlabelled in any array called hashes

A document hash or other value subject to cryptographic authentication

The hash object may have a (string) property

  • algorithm property (optional)

When appearing as part of a resource (data object), the hash should apply to the referenced object.

This object appears as a property on: hlink.
