Organization: org object

An organization or legal entity (not a person), with contact information

The org object has the following members (properties):

  • org-name object (optional)
  • short-name object (optional)
  • An array labelled organization-identifiers, containing org-id objects (optional)
  • address object (optional)
  • An array labelled email-addresses, containing email objects (optional)
  • An array labelled telephone-numbers, containing phone objects (optional)
  • An array labelled URLs, containing url objects (optional)
  • notes object (optional)

This object appears as a property on: party.
