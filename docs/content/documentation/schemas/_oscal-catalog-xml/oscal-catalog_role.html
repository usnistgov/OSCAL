Role: role element

Defining a role to be assigned to a party or agent

The role element has the following contents (in order):

  • id attribute (required)
  • title element (zero or one)
  • short-name element (zero or one)
  • desc element (zero or one)

Permissible values to be determined closer to the application (e.g. by a receiving authority).

This element appears inside: metadata.
