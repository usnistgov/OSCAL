Hashed link: hlink element

A pointer to an external document with optional hash for verification

The hlink element has the following contents (in order):

  • href attribute (required)
  • rel attribute (optional)
  • media-type attribute (optional)
  • title element (zero or one)
  • hash element (zero or more)

This is different from unstructured link, which makes no provision for a hash or formal title. It is also different from the element of the same name once proposed for HTML.

This element appears inside: metadata, resource.
