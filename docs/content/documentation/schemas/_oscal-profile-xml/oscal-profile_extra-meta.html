Extra metadata: extra-meta element

Container for (system-defined) metadata

The extra-meta element has the following contents (in order):

  • A choice between
    • meta-group element (zero or more)
    • meta element (zero or more)
  • notes element (zero or one)

This element appears inside: metadata.
