Match controls and subcontrols by identifier: match element

Select controls by (regular expression) match on ID

The match element may have attributes

  • pattern attribute (optional)
  • order attribute (optional)
  • with-control attribute (optional)
  • with-subcontrols attribute (optional)

This element appears inside: custom, group, include, exclude.
