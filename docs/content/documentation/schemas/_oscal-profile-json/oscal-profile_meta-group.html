Metadata group: meta-group object

This object appears unlabelled in any array called metadata-groups

Custom- or application-defined annotated metadata.

The meta-group object has the following members (properties):

  • term property (optional)
  • type property (optional)
  • An array labelled metadata-fields, containing meta objects (optional)
  • An array labelled metadata-groups, containing meta-group objects (optional)
  • notes object (optional)

This object appears as a property on: extra-meta, meta-group.
