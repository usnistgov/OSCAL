Publication metadata: metadata object

metadata is the root (containing) object of this schema.

Information describing the publication or availability of this document

The metadata object has the following members (properties):

  • title object (required)
  • An array labelled authors, containing author objects (optional)
  • publication-date object (optional)
  • version object (optional)
  • An array labelled document-identifiers, containing doc-id objects (optional)
  • An array labelled properties, containing prop objects (optional)
  • An array labelled hashed-links, containing hlink objects (optional)
  • An array labelled resources, containing resource objects (optional)
  • An array labelled roles, containing role objects (optional)
  • An array labelled parties, containing party objects (optional)
  • notes object (optional)
  • extra-meta object (optional)
