Address: address object

A postal address

The address object has the following members (properties):

  • An array labelled postal-address, containing addr-line objects (optional)
  • city object (optional)
  • state object (optional)
  • postal-code object (optional)
  • country object (optional)

This object appears as a property on: person, org.
