Addition: add object

This object appears unlabelled in any array called additions

Specifies contents to be added into controls or subcontrols, in resolution

The add object has the following members (properties):

  • position property (optional)
  • title object (optional)
  • An array labelled parameters, containing param objects (optional)
  • An array labelled props, containing prop objects (optional)
  • An array labelled links, containing link objects (optional)
  • An array labelled parts, containing part objects (optional)
  • references object (optional)

This object appears as a property on: alter.
