Type: type object

Object of type string

Indicating the type of identifier or other data item.

Generally, this flag is used to relate to schemes and taxonomies defined outside the OSCAL application in some public form; i.e. they can be looked up.

This object appears as a property on: doc-id, person-id, org-id, resource, meta-group, meta.
