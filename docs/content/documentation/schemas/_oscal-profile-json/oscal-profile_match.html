Match controls and subcontrols by identifier: match string object

This object appears unlabelled in any array called pattern-selectors

Select controls by (regular expression) match on ID

The match object may have (string) properties

  • pattern property (optional)
  • order property (optional)
  • with-control property (optional)
  • with-subcontrols property (optional)

This object appears as a property on: custom, group, include, exclude.
