Reference: ref object

This object appears unlabelled in any array called refs

A reference, with one or more citations to standards, related documents, or other resources

The ref object has the following members (properties):

  • id property (optional)
  • An array labelled citations, containing citation objects (optional)
  • Prose contents (paragraphs and lists)

This object appears as a property on: references.
